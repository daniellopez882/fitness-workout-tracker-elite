# fitness-workout-tracker-elite

A comprehensive fitness and workout logging application designed to help users track their progress, manage exercises, and achieve their health goals. Built with Django and React, this app provides an "elite" experience for managing your fitness journey.

## Core Features
- **Workout Logging**: Track your daily workouts with detailed exercise logs.
- **Exercise Database**: A pre-loaded list of exercises with the ability to add custom ones.
- **Progress Tracking**: Visualize your fitness data and track your improvements over time.
- **Secure Auth**: JWT-based authentication for data privacy.

## Tech Stack
- **Backend**: Django, Django REST Framework, PostgreSQL.
- **Frontend**: React, React Bootstrap, React Query, Redux.
- **Deployment**: Dockerized for easy deployment (Heroku ready).

## Setup Guide
1. **Local Setup**: Run `docker-compose up --build` to spin up the entire stack.
2. **Access**:
   - Frontend: `http://localhost:3000`
   - API Docs: `http://localhost:8000/swagger/`
